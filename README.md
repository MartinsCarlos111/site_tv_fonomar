# Site TV Fonomar

Frontend desenvolvido com [Next.js](https://nextjs.org) para simulação e contratação de planos de anúncio da TV Fonomar.

## Funcionalidades

- Formulário de simulação de planos de anúncio
- Seleção de locais de exibição, fidelidade e plano
- Cálculo automático do valor mensal com base nas escolhas
- Envio dos dados via Google Apps Script

## Tecnologias

- [Next.js 16](https://nextjs.org) (App Router)
- [React 19](https://react.dev)
- TypeScript
- CSS Global (sem framework CSS)

## Como rodar localmente

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000) no navegador.

## Scripts disponíveis

| Comando | Descrição |
|---------|-----------|
| `npm run dev` | Inicia o servidor de desenvolvimento |
| `npm run build` | Gera o build de produção |
| `npm run start` | Inicia o servidor de produção |
| `npm run lint` | Executa o linter (ESLint) |
